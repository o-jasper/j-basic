macro alist

arguments: (&rest vars)

Put in values, this will make a alist with the indicating symbols based on the code of given values. If the values are variables, the variable name will be given, if it is an s-expression, the first argument is used. `(:a key value)` explicitly sets the key. Otherwise, it produces an (macroexpand-time)error. If the first argument is a keyword, it will make all the symbols for the plist that package. (For instance `:keyword` to turn it into a 'regular' plist)
