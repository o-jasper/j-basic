package read-tab-listing

Reading and writing nested lists using indentation, and conversion between s/m-expressions. Note that it can't do two consecutive lists, m-expressions are a bit of a work-around, but that it cannot do expressions without arguments. Made initially to read the output if the wireless internet connection lister, iwlist. TODO allow it to screen out comments.
